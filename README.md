# 🌗 Responsive Blog Page with Theme Toggle | Task 8

This project is a fully responsive **blog layout** built using **Bootstrap 5** with a built-in **dark/light mode toggle**. It was created as part of **Task 8** in the Elevate Labs (Web Development) Internship.

---

## 📌 Features

- 🌗 Dark / Light **theme toggle** (with animated icon)
- 🔍 **Search bar** in the navbar
- 📸 Responsive **blog cards** with hover effects
- 🏷️ **Date badges** and author profiles with avatar
- 📰 **Blog preview modal** using Bootstrap
- 📂 **Category filter buttons**
- 📬 Newsletter subscription form
- 🧭 Sticky, responsive **navigation bar**
- ⬆️ **Scroll to top** button
- 📆 **Dynamic current year** in footer
- 🎨 Designed with **Bootstrap utility classes** only
